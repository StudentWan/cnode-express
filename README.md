#cnodexpress
This is a practice project, developed with Node+ Express+ Handlebars.

Mainly technology stacks:

    superagent

    express-Handlebars

    express-formidable

    express-session

    config-lite
##Install

npm install

##Run
If you installed supervisor, then type:

    supervisor --harmony index.js

If not, type:

    node index.js
